# 🚀 Gravewave

**Gravewave** is a high-octane, space-themed survival game built with **Vanilla JavaScript** and the **HTML5 Canvas API**. Navigate your rocket through a treacherous field of procedurally generated, jagged asteroids and survive the cosmic chaos.

-----

## 🛠 Features

  * **Physics-Based Movement:** Smooth rocket handling and rotation logic.
  * **Procedural Asteroids:** Uses trigonometry to generate unique, jagged rocky shapes rather than simple circles.
  * **Collision Detection:** Precise hit-box logic between the rocket and oncoming meteors.
  * **Zero Frameworks:** Written in pure Vanilla JS for maximum performance and a lightweight footprint.

## 🕹 How to Play

1.  **Launch the Game:** Open `index.html` in any modern web browser.
2.  **Controls:**
      * **Spacebar:** Primary action/thrust.
      * **Arrow Keys / WASD:** Navigate your rocket.
3.  **Objective:** Dodge the incoming meteors for as long as possible to increase your high score.

## 📂 Project Structure

  * `index.html` - The entry point and Canvas container.
  * `gravewave.js` - The core game loop, physics, and rendering logic.
  * `gravewave.css` - Styling for the game container and UI elements.

  * **HTML5 Canvas** - For 2D rendering.

  * **CSS3** - Layout and space-themed aesthetics.
